# World

This folder holds the public, PC-eye view of the Underleaf setting — what your characters know about the world they live in at the start of the campaign. It is safe to browse.

The deeper layers of the setting — the actual cosmology, the magic system, the eventual antagonist — are not here. They live in [`design/DM-ONLY/`](../design/) and are revealed through play, not by reading ahead.

## Files

- [overview.md](overview.md) — what anyone in the world knows.
- `places/` — specific locations as they are added.

## For the DM

The PC-eye view is a useful authoring constraint: any time you write something in `world/`, ask whether the PCs could plausibly know it. If yes, it goes here. If no, it goes in `design/DM-ONLY/` or in an episode's `dm/` subfolder.

This separation makes the public files browsable by players without spoiling anything, and the DM-only files are clearly marked.
