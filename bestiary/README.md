# Bestiary

This folder holds threat / creature / antagonist records. Underleaf is sparing with bestiary entries — most threats in the campaign are *people*, and the campaign's central antagonist is not the kind of thing that fits a stat block.

Bestiary entries that exist mostly describe environmental dangers, animals, and the occasional anomaly.

Player note: encountering a threat in play is one thing; reading the bestiary cold is a self-spoiler. The Quire runtime surfaces entries only after they appear in a scene.
