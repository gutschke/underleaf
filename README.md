# Underleaf

Sample campaign for the [Quire](https://github.com/gutschke/quire) TTRPG framework.

> **Status: in development.** Not yet playable. This README is a placeholder.

## Setting

Underleaf is a contemporary-feeling original world with a hidden depth. Cyberpunk-adjacent in atmosphere (Gibson / Sterling / Stephenson), explicitly non-dystopian. Story-forward, theater-of-the-mind, 15+.

The world's deeper mechanics are revealed gradually during play, not info-dumped. Background reading for prospective DMs will land in `design/` during phase 0.

## Playing

Once Quire reaches phase 1 of development, this campaign will be playable at:

<https://play.quire.games?campaign=gutschke/underleaf>

## Forking

Underleaf demonstrates the first of three forking modes for Quire:

1. **Fork content** (what this campaign is) — copy this repo, edit, players visit `play.quire.games?campaign=your-name/your-fork`. No hosting setup; uses the canonical runtime.
2. **Fork content + self-host runtime** — see [Quire](https://github.com/gutschke/quire) for the documented self-host path.
3. **Full re-skin** — fork both, change names, run your own variant.
