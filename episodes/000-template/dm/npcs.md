# NPCs — Template Episode

Players ask for NPC names at the worst possible moments. This file exists so the DM has an answer.

**None of this is load-bearing.** Every entry is optional reference material. Use what's helpful; ignore what isn't. See [Quire's authoring conventions](https://github.com/gutschke/quire/blob/main/design/authoring.md#ship-an-npcs-quick-reference) for the full pattern.

## How to use this file

- Skim once before the session. Don't memorize.
- When an NPC becomes recurring, promote them to `characters/npcs/<id>.json` under the campaign root.
- Diversity matters. Names and identities should reflect the setting plausibly.
